## Installation

Install the package using your preferred package manager:

```bash
npm install my-package
```

Or with Bun:

```bash
bun add my-package
```

## Basic Usage

Import the module and call the `init` function to get started:

```typescript
import { init } from 'my-package'

const app = init({
  name: 'my-app',
  debug: true,
})

app.start()
```

<Note>The `debug` option enables verbose logging. Disable it in production.</Note>

## Configuration

Create a configuration file at the root of your project:

```json
{
  "name": "my-app",
  "version": "1.0.0",
  "features": {
    "analytics": true,
    "notifications": false
  }
}
```
